# Codex
A simple Web3 learning platform where you can study WEB3 Lectures of diffrent chains, practice coding smart contracts, and improve your skills with a built-in ranking system.


### Tech Stack
- Next.js  
- Bun  
- Solidity  


### Run Locally

```bash
# Clone the repo
https://github.com/anukulKun/Codex.git

cd Codex

# Install dependencies
bun install

# Start the dev server
bun dev
